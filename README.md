# Domain

This is a 3D survival game built with the Godot Engine. It uses procedural generation and is based on games like 'Muck' and 'Minecraft'.

## Engine

This project uses:
- Godot Engine (recommended version: 4.x)

Download: https://godotengine.org

---

## How to Run

### Option 1: Open in Godot
1. Install Godot Engine
2. Clone or download this repository
3. Open `project.godot` in Godot
4. Press **Play (F5)**

---

### Option 2: Play Exported Builds
If releases are available, download the correct build for your system:

- Windows: `.exe`
- Linux: `.x86_64`
- macOS: `.app`
- Web: open `index.html` in a browser or use hosted version
