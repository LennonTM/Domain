# Domain

This is a 3D survival game built with the Godot Engine. It uses procedural generation and is based on games like 'Muck' and 'Minecraft'.

## Engine

This project uses:
- Godot Engine (recommended version: 4.6.2-stable)

Download: https://godotengine.org

---

## How to Run

### Open in Godot
1. Install Godot Engine
2. Clone or download this repository
3. Open `project.godot` in Godot
4. Press **Play (F5)**
